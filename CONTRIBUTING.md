# Contributing to F1 Hub Mobile

Terima kasih sudah tertarik berkontribusi! 🏎️

## 🐛 Lapor Bug

1. Cek [Issues](../../issues) — pastiin belum ada yang lapor hal yang sama
2. Buat issue baru dengan template **Bug Report**
3. Sertakan: versi APK, tipe HP, langkah reproduksi, screenshot

## 💡 Request Fitur

1. Buat issue dengan label `enhancement`
2. Jelaskan fitur yang diinginkan dan kenapa berguna

## 🔧 Pull Request

1. Fork repo ini
2. Buat branch baru: `git checkout -b fitur/nama-fitur`
3. Commit dengan pesan jelas: `git commit -m "feat: tambah fitur X"`
4. Push dan buat Pull Request

## 📝 Konvensi Commit

```
feat:     Fitur baru
fix:      Bug fix
style:    Perubahan UI/styling
refactor: Refactoring kode
docs:     Update dokumentasi
chore:    Update dependency / config
```

## 🛠️ Setup Dev

```bash
git clone https://github.com/leviar/f1-hub-mobile.git
cd f1-hub-mobile
npm install
npx expo start
```

Scan QR dengan Expo Go di HP untuk testing.
