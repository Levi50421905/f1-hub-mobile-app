## [1.0.0] — 2026-03-31

### 🎉 Rilis Pertama

#### Home
- Next Race Card dengan countdown live (DD:HH:MM:SS)
- Cuaca sirkuit real-time via Open-Meteo API
- Driver Standings top 5 dengan warna tim
- Race berikutnya dengan jadwal WIB/WITA/WIT
- F1 Glossary preview dengan tombol Lihat Semua

#### Klasemen
- Driver Standings & Constructor Standings musim 2026
- Data real-time dari Jolpica API
- Bendera negara setiap driver & tim
- Highlight driver favorit ⭐

#### Kalender
- Jadwal lengkap 23 race musim 2026
- Filter: Semua / Akan Datang / Selesai
- Semua circuit bisa di-tap untuk detail

#### Race Detail
- Jadwal semua sesi (FP1, FP2, FP3, Qualifying, Race)
- Hasil Race dengan podium 🥇🥈🥉
- Hasil Qualifying dengan waktu Q1/Q2/Q3
- Hasil Sprint (hanya tampil di sprint weekend)
- Cuaca sirkuit real-time
- Animated header — background muncul saat scroll
- Tombol back langsung ke Kalender

#### Driver
- Profil lengkap semua driver 2026
- Statistik karier: total race, wins, podium, pole position, fastest lap, gelar dunia
- Animated hero yang fade saat scroll
- Set driver favorit ⭐

#### F1 Glossary
- 20+ istilah F1 dalam Bahasa Indonesia & English
- Search real-time
- Dapat diakses dari Home atau tab Glossary

#### Setting
- 4 Tema: Dark, Carbon, Light Sports, Neon
- Race Weekend Mode — tema berubah otomatis saat race weekend
- Timezone: WIB / WITA / WIT
- Bahasa: Indonesia / English
- Notifikasi: pengingat 30 menit sebelum FP1, FP2, FP3, Qualifying, Sprint, Race
- Driver favorit
